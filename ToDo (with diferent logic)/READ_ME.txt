This app is task from educational course.

CSS file was given.

It was necessary to solve the following problems:

1) Draw all to do tasks using JavaScript code.

2) Implement new task creation logic.

3) Add input validation

4) Create a modal window where the user accepts removing a task from the list.

5) Implement a change of theme from dark to light and vice versa. 
   The theme should switch when you push the "Tab" button. 
   Initially the theme is light.
