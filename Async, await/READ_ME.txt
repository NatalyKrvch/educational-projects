Task description: 

You need to create a function that will display album data in the DOM tree. 
For data, use the following url: https://jsonplaceholder.typicode.com/albums

For convenience, add a span element with the text "Loading..." before downloading albums, 
and hide this element after downloading the data.
Use async await and try, catch, finally. 
If an error occurred while loading albums, put the text "Error in getting the album's data." 
inside the tag with the class equal to "data-container".

HTML and CSS are given.
