Task description: 

Imagine you have a new project that has the following functionality:
displaying the photo that downloaded the fastest.
Create a function that accepts 1 ids parameter, which is an array of id parameters on the photo object. 
To get information about the photo, you need to use the following URL: https://jsonplaceholder.typicode.com/photos  
With the help of an array of ids get the data about the photo that was most downloaded at fetch request
For convenience, add a span element with the text "Loading..." before uploading the photo, 
and hide this element after downloading the data.

HTML and CSS code are given.
