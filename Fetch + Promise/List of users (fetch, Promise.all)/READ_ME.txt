Task description: 

You need to create a function that accepts an array of the users ids. 
You need to get all users whose id values are the same as in the given array: [5, 6, 2, 1].
Use this address to get user information: https://jsonplaceholder.typicode.com/users.

For convenience, add a span element with the text "Loading..." before loading users, 
and hide this element after downloading user data.

HTML and CSS code are given.
